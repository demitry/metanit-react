Если рассмотреть поэтапно, то рендеринг компонента будет происходить следующим образом:
Для рендеринга компонента вызывается функция ReactDOM.render()
React вызывает компонент Hello
Компонент Hello возвращает элемент Привет, Eugene
React DOM обновляет стандартный DOM веб-страницы, чтобы она содержала элемент Привет, Eugene
